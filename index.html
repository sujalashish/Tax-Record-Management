Developer Setup & Running Guide

Follow these quick commands to build and run the desktop app locally on your machine.

1

Install Prerequisites

Ensure you have **Java JDK 26** and **Maven** installed, along with a running instance of **MySQL Server**.

2

Database Schema Setup

Create a database named `trm_db` in MySQL and configure user credentials inside the `DBConnection.java` configuration class.

3

Compile & Exec

Run the Maven compilation cycle and launch the main class using the execution script configuration.

PowerShell — TRM_Sujal
— ☐ ✕

PS> git clone https://github.com/sujalashish/Tax-Record-Management.git

PS> cd Tax-Record-Management

PS> mvn clean compile

PS> mvn exec:java
